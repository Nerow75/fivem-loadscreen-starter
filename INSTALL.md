# 🚀 Installation Rapide

## 1️⃣ Extraire l'archive

```bash
tar -xzf loadscreen-fivem.tar.gz
cd loadscreen-fivem
```

## 2️⃣ Installer les dépendances

```bash
npm install
```

## 3️⃣ Personnaliser

- Remplacez `public/logo.png` par votre logo
- Éditez `src/App.jsx` pour les URLs vidéos/musique et nom du serveur

## 4️⃣ Builder

```bash
npm run build
```

## 5️⃣ Déployer sur FiveM

Copiez le dossier complet dans `resources/` et ajoutez dans `server.cfg`:

```
ensure loadscreen-fivem
```

✅ C'est prêt !
